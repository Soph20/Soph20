## What I'm building

**[Mnemos](https://github.com/Soph20/mnemos-capture)**  — A knowledge pipeline for agentic workflows. Paste any resource — articles, threads, transcripts, ideas — and an LLM extracts the insight, tags it, and commits it to a GitHub repo you own. Your AI agents pull from it via MCP or Git. Built with Haiku 4.5, prompt caching, and BYOK — costs under $1/month. Try it: [here](https://github.com/Soph20/mnemos-capture)

**Promix OS** — A runtime that turns Claude Code into a team of 15 specialized AI agents — PM, Designer, Architect, Engineer, QA, Legal Counsel, and more. Each agent runs as its own Claude instance with its own context, reasoning, and tools. The PM can't write code. The Engineer can't expand scope. QA can block a release. Hard boundaries, not role-play.

A Chief of Staff orchestrator (Promix) sequences the work, enforces quality gates, and keeps everyone focused. No code ships without QA evidence. No engineering starts without PM sign-off. 121+ skills across 13 domains — from contract risk analysis to Stripe payment architecture to accessibility reviews. Decisions and learnings persist in a memory vault, so the system doesn't repeat mistakes across sessions.

Built ADHD-aware: when it detects cognitive overload (too many ideas, topic switching, emotional urgency), it forces a single objective and parks the rest. One workforce powers multiple products, each with its own repo and stack, sharing the same agents and memory.

**Nomos** — Agentic workflow engine for product managers. Discovery, delivery, stakeholder comms, sprint planning — all running inside Claude Code with persistent memory and MCP integrations. Your PM operating system.
